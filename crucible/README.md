# Crucible

The **planning brain** of a three-skill trio:

> **ResearchOne** (research) → **Crucible** (planning) → **Foreman** (build)

Crucible turns an intent — or a messy existing project — into a **vetted, gold-standard, Foreman-ready implementation plan**, through structured brainstorming, a best-in-class restatement, and bounded multi-round adversarial refinement (debate + ResearchOne + critique), all while refusing to drift from the project's locked objective.

- **Status:** design phase (2026-06-04). See `MASTER-PLAN.md`.
- **Decisions:** `DECISION-LOG.md`.
- **Orientation for agents:** `CLAUDE.md`.

Production-only. Built to be merge-able into Project-Manager later, alongside Foreman.
